### Installing Android Studio

Install from link: https://developer.android.com/studio

---

### Setting Up Virtual Device

1. New Project -> Name -> Package Name -> Activity Type
2. Click on Device Manager -> New Device -> Hardware -> Software -> Finish
3. Run boilerplate on AVD for testing
